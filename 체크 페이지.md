컴퓨텨에게 명령하는 말(파이썬) 처음 배우고 작업 이력 남기기

**과제 목표**
VSCode에서 Python 파일을 생성하고 실행
터미널에서 Python/Git 버전을 확인하고 필요한 설정 점검
파이썬 기초 문법 설명
Git이 무엇이고, 왜 필요한지 설명
init, add, commit, push, pull, checkout, clone, merge가 각각 무엇인지 설명
브랜치 생성 및 병합
Github 내 코드 업로드 및 관리

**요구 사항**
1) 개발 환경
   VSCode 내 Python 확장(Extension) 설치 + Korean Language Pack 설치 (선택)
   터미널에서 Python 3.10 이상인지 확인
   간단한 print("Hello") 코드를 작성 후 실행
   Git 버전 확인 + 사용자 정보(이름, 이메일) 설정
   기본 브랜치 이름을 main으로 설정
   VSCode에서 Github 계정으로 로그인하고 연동이 정상정으로 되었는지 확인

2) Git 저장소 설정 및 초기화
   Github에 새 저장소를 생성
   프로젝트 폴더를 만들고 로컬에서 init으로 저장소를 시작
   원격 저장소를 연결하고 첫 커밋과 푸시를 수행 (add, commit, push)
   .gitignore 파일을 생성하여 불필요한 파일을 제외
   README.md 파일을 생성하고 프로젝트 제목을 작성
   공개 샘플 저장소 1개를 clone으로 내려받아 폴더 구조와 로그를 확인 (확인 후 삭제 가능)

3) 프로그램 실행 및 메뉴
   프로그램 실행 시, 메뉴 출력
   사용자가 번호를 입력하여 기능을 선택
   잘못된 번호 입력 시 안내 메시지를 출력하고 다시 메뉴를 보여줌

4) 기본 프롬프트 데이터
   프로그램 시작 시 이전 미션에서 작성한 프롬프트가 최소 3개 이상 미리 등록되어 있어야함
   프롬프트 데이터는 리스트와 딕셔너리를 사용하여 저장
   각 프롬프트는 제목, 내용, 카테고리, 즐겨찾기 여부를 포함
   카테고리 예시: 텍스트 생성, 이미지 생성, 영상 생성, 페르소나, 자동화, 기타

5) 프롬프트 추가
   새로운 프롬프트 등록 가능
   입력 항목은 제목, 내용, 카테고리
   입력값이 비어있으면 다시 입력 요청
   카테고리는 미리 정의된 목록에서 선택하거나 직접 입력
   추가된 프롬프트는 리스트에 저장 (프로그램 실행 중에만 유지)
   즐겨찾기 기본값은 False

6) 프롬프트 목록 (브랜치 활용)
   main 브랜치 이외의 추가 브랜치를 생성하고 이 기능을 해당 브랜치에서 작업
   저장된 모든 프롬프트를 번호와 함께 출력
   각 프롬프트의 제목, 카테고리, 즐겨찾기 여부(별) 표시
   프롬프트가 없으면 안내 메시지 출력
   기능 완성 후 커밋하고 main 브랜치로 병합 (checkout, merge)

7) 카테고리별 조회
   카테고리 목록을 보여주고 선택하면 해당 카테고리의 프롬프트만 출력
   해당 카테고리에 프롬프트가 없으면 안내 메시지 출력

8) 프롬프트 검색
   키워드를 입력받아 제목 또는 내용에 포함된 프롬프트를 검색
   검색 결과를 목록으로 출력
   검색 결과가 없으면 안내 메시지 출력

9) 프롬프트 상세 보기
   프롬프트 번호를 입력하면 해당 프롬프트의 전체 내용을 출력
   제목, 카테고리, 즐겨찾기 여부, 내용 전체를 표시
   잘못된 번호 입력 시 안내 메시지 출력

10) 즐겨찾기 관리
    프롬프트 번호를 입력하여 즐겨찾기를 추가/해제
    즐겨찾기된 프롬프트만 모아서 확인 가능

11) 코드 구조
    필수 요구사항은 외부 라이브러리 없이 기본 문법만 사용
    모든 코드를 한 함수에 몰아 넣지 않을 것
    기능별로 함수 분리
    ex) show_menu(), add_prompt(), show_list(), search_prompt(), show_favorites()

13) README.md 작성
    프로그램 이름과 간단한 설명을 작성
    실행 방법 작성
    기능 목록 작성
    등록된 프롬프트 카테고리 설명

1. 동작하는 프롬프트 관리 프로그램 (Python 콘솔 프로그램)
```
터미널에서 메뉴 번호를 입력해 기능을 선택하는 콘솔 기반 프로그램
프롬프트 추가, 목록 보기, 카테고리별 조회, 검색, 상세 보기, 즐겨찾기 기능 등 동작
프로그램 실행 중 추가한 프롬프트와 즐겨찾기 상태 유지 (종료 시 초기화)
이전 미션에서 작성한 프롬프트가 기본 데이터로 최소 3개 이상 등록
```
